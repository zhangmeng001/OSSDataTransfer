# 介绍

OSSDataTransfer是一个对象存储数据管理工具，可用于以下几个场景：
1. 将数据迁移到对象存储，涉及到数据的备份、迁移等；
2. 备份数据的还原；
3. 数据的下载；

>目前只支持亚马逊S3产品，其他产品后面会陆续支持。
# 前提

需要jdk1.7+

# 使用
